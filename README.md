Momentum Online Coaching

Moderní landing page pro online fitness coaching vytvořená v HTML, CSS a JavaScriptu.

Cílem projektu bylo navrhnout a naprogramovat profesionálně působící web zaměřený na prezentaci online trenérských služeb. Zaměřil jsem se především na moderní design, responzivitu a přehlednou strukturu jednostránkového webu.

Co projekt obsahuje

* moderní one-page layout
* responzivní design pro mobilní zařízení i desktop
* navigaci se scrollováním mezi sekcemi
* hero sekci s výzvou k akci
* sekci služeb a benefitů coachingu
* kontaktní sekci
* vlastní animace a interaktivní prvky
* čistý a přehledný UI design

Technologie

* HTML
* CSS
* JavaScript
* Git
* GitHub Pages

Co jsem se na projektu naučil

* vytvářet moderní landing pages
* pracovat s rozvržením one-page webů
* navrhovat responzivní uživatelské rozhraní
* lépe organizovat strukturu HTML a CSS
* vytvářet plynulé scrollování mezi sekcemi
* zlepšovat celkový uživatelský zážitek (UX)
* publikovat projekty pomocí GitHub Pages
